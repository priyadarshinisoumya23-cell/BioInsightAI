# 🧬 BioInsight AI

BioInsight AI is a bioinformatics dashboard built with Streamlit for analyzing gene expression datasets.

## Features

- 📂 Data Explorer
- 🔍 Gene Search
- 📊 Interactive Visualizations
- 🤖 Machine Learning Analysis
- 🧬 Differential Gene Expression
- 📄 PDF Report Generation
- 📥 CSV Download

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Matplotlib
- Seaborn
- Scikit-learn
- FPDF2

## Installation

```bash
git clone <your-repository-url>
cd BioInsightAI
pip install -r requirements.txt
streamlit run app.py
```
